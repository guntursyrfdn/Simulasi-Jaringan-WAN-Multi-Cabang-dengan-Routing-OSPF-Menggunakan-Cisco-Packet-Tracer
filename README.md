# Simulasi-Jaringan-WAN-Multi-Cabang-dengan-Routing-OSPF-Menggunakan-Cisco-Packet-Tracer
Simulasi jaringan WAN perusahaan xyz.net menggunakan Cisco Packet Tracer. Mencakup routing dinamis OSPF, konfigurasi DHCP di router cabang, serta DNS dan Web Server di kantor pusat dengan domain wali.id. Diuji dengan ping dan traceroute antar semua cabang.
